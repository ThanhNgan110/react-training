## How to run

- Clone source

```
git clone https://github.com/ThanhNgan110/react-training
```

- Git checkout practice-two

```
git checkout practice-two
```

- Go to folder practice-two

```
cd practice-two
```

- Install packages

```
pnpm install
```

- Run project

```
pnpm run dev
```
